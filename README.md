# HabitFlow Web

Projeto web do HabitFlow, desenvolvido em Next.js.

## Scripts

- `npm install` – instala dependências
- `npm run dev` – roda em desenvolvimento
- `npm run build` – build de produção
- `npm start` – roda o build

## Estrutura principal

- `src/app` – rotas (públicas e autenticadas)
- `src/components` – componentes compartilhados
- `src/lib/designSystem.js` – tokens do design system
- `src/context/AuthContext.jsx` – contexto de autenticação (stub inicial)
